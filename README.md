# 360° Marketing Intelligence Dashboard 
> Python · Power BI

## 📌Problem Statement
Marketing teams often struggle to measure campaign effectiveness because customer, transaction, and marketing data are spread across multiple sources. This lack of a unified view makes it difficult to analyze conversion rates, customer lifetime value (LTV), product performance, and overall marketing ROI. This project addresses these challenges by building an interactive Power BI dashboard that consolidates key metrics into actionable insights, enabling data-driven marketing decisions and performance optimization.
